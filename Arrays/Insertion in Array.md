### Insertion in Array

we use insert() function which takes 2 arguments.

Syntax : <array_name>.insert(index,element)
  
 ```python
  
from array import*
array_name = array('i',[100,90,80,70,60,50])
array_name.insert(2,20)

for x in array_name:
    print(x)
```
Output:
```
100
90
20
80
70
60
50
  ```
